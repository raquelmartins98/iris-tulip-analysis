# Iris + Tulip Mania Analysis

Análisis de datos del Iris dataset con comparación histórica de la Tulip Mania (1637).

## Contenido

- `Iris.csv` - Dataset Iris con 150 flores
- `iris_analisis.py` - Script de análisis en Python
- `README.md` - Documentación

## Iris Dataset

Dataset clásico de 150 flores Iris con 4 características:
- SepalLengthCm
- SepalWidthCm  
- PetalLengthCm
- PetalWidthCm

3 especies: setosa, versicolor, virginica

## Comparación con Tulip Mania

| Aspecto | Tulip Mania (1637) | Iris Dataset (1936) |
|---------|------------------|-------------------|
| Tipo | Primera burbuja especulativa | Dataset fundacional ML |
| Creador | Mercados holandeses | Ronald Fisher |
| Legacy | Metáfora de burbujas financieras | "Hello World" de ML |
